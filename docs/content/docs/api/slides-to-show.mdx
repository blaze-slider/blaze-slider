Number of slides visible in the slider at a time.

```ts
slidesToShow: number; // default: 1
```

Blaze Slider sets this value as the `--slides-to-show` CSS variable on the slider element.

<Callout>
  Set `--slides-to-show` in CSS to prevent layout shifts. See [Preventing Layout
  Shifts](/docs/tutorial/layout-shift).
</Callout>

## Example: 1 slide (default)

```js
new BlazeSlider(el, {
  all: { slidesToShow: 1 },
});
```

<ExampleSlider addNav config={{ all: { slidesToShow: 1 } }}>

  <div>2</div>
  <div>3</div>
  <div>4</div>
  <div>5</div>
  <div>6</div>
</ExampleSlider>

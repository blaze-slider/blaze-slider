---
title: Framework Integration
description: Use Blaze Slider with React, Vue, Svelte, and other frameworks
---

Blaze Slider is framework agnostic. It just needs a DOM element and an optional config object.

Every framework provides a way to access the underlying DOM element:

- React: [Ref API](https://react.dev/learn/referencing-values-with-refs)
- Vue: [Template Refs](https://vuejs.org/guide/essentials/template-refs.html)
- Svelte: [bind:this](https://svelte.dev/docs#template-syntax-element-directives-bind-this)
- Solid: [ref](https://www.solidjs.com/docs/latest/api#ref)

Use these APIs to get the element reference, then initialize with `new BlazeSlider(element, config)`.

<Callout type="warn">
  Don't initialize BlazeSlider on the same element more than once.
</Callout>

## Example

### React

```js title="useBlazeSlider.js"
import { useRef, useEffect } from 'react';
import BlazeSlider from 'blaze-slider';
import 'blaze-slider/dist/blaze.css';

export function useBlazeSlider(config) {
  const sliderRef = useRef(null);
  const elRef = useRef(null);

  useEffect(() => {
    if (!sliderRef.current && elRef.current) {
      sliderRef.current = new BlazeSlider(elRef.current, config);
    }
  }, []);

  return elRef;
}
```

```jsx title="App.jsx"
import { useBlazeSlider } from './useBlazeSlider';

function App() {
  const ref = useBlazeSlider({
    all: { slidesToShow: 3 },
  });

  return (
    <div className="blaze-slider" ref={ref}>
      <div className="blaze-container">
        <div className="blaze-track-container">
          <div className="blaze-track">
            <div>1</div>
            <div>2</div>
            <div>3</div>
            <div>4</div>
          </div>
        </div>
      </div>
    </div>
  );
}
```

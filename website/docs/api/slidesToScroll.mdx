# `slidesToScroll`

The number of slides to scroll when the user navigates to the previous or next page by either swiping, dragging or using the previous and next navigation buttons.

## default

```typescript
slidesToScroll: 1
```

<br />

### Example: `slidesToScroll: 1`

```javascript
new BlazeSlider(el, {
  all: {
    slidesToShow: 3,
    slidesToScroll: 1, // ( default )
  },
})
```

<ExampleSlider
  addNav={true}
  config={{
    all: {
      slidesToShow: 3,
      slidesToScroll: 1,
    },
  }}
>
  <div> 1 </div>
  <div> 2 </div>
  <div> 3 </div>
  <div> 4 </div>
  <div> 5 </div>
  <div> 6 </div>
</ExampleSlider>

<br />

### Example: `slidesToScroll: 3`

```javascript
new BlazeSlider(el, {
  all: {
    slidesToShow: 3,
    slidesToScroll: 3,
  },
})
```

<ExampleSlider
  addNav={true}
  config={{
    all: {
      slidesToShow: 3,
      slidesToScroll: 3,
    },
  }}
>
  <div> 1 </div>

  <div> 6 </div>

:::note
Notice how the number of pages decreased compared to the previous example, because this slider is scrolling 3 slides at a time.

:::

<hr />

## Edge Cases

If there are not enough slides to scroll, the Blaze slider will dynamically reduce the `slidesToScroll` value.
Let's take a look at the slider with 7 slides and `slidesToShow: 3`

### Example (loop enabled)

if you scroll next on the last page (to go to the first page), only 1 slide is scrolled instead of 3

```javascript
new BlazeSlider(el, {
  all: {
    slidesToShow: 3,
    slidesToScroll: 3,
  },
})
```

<ExampleSlider
  addNav={true}
  config={{
    all: {
      slidesToShow: 3,
      slidesToScroll: 3,
    },
  }}
>
  <div> 1 </div>
  <div> 2 </div>
  <div> 3 </div>
  <div> 4 </div>
  <div> 5 </div>
  <div> 6 </div>
  <div> 7 </div>
</ExampleSlider>

### Example (loop disabled)

if you scroll next on the second page, only 1 slide is scrolled instead of 3

```javascript
new BlazeSlider(el, {
  all: {
    loop: false,
    slidesToShow: 3,
    slidesToScroll: 3,
  },
})
```

<ExampleSlider
  addNav={true}
  config={{
    all: {
      loop: false,
      slidesToShow: 3,
      slidesToScroll: 3,
    },
  }}
>
  <div> 1 </div>
  <div> 2 </div>
  <div> 3 </div>
  <div> 4 </div>
  <div> 5 </div>
  <div> 6 </div>
  <div> 7 </div>
</ExampleSlider>
